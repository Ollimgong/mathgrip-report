📊 구글 시트 연동 한 번만 설정하면 매달 클릭 한 번!
1
구글 시트 "웹에 게시" 설정
구글 시트 열기 → 파일 → 공유 → 웹에 게시 → 게시 클릭
⚠️ 이 설정을 해야 앱에서 시트 데이터를 읽을 수 있어요. 시트 내용은 링크를 아는 사람만 볼 수 있습니다.
2
Apps Script URL 입력
배포 완료 후 받은 웹 앱 URL을 붙여넣기
구글 시트 ID 입력
시트 주소에서 ID를 복사해서 붙여넣기
.../spreadsheets/d/여기가_시트_ID/edit
3
시트 탭 이름 규칙
탭 이름을 YYYY-MM 형식으로 만들어야 자동으로 인식돼요
2026-03 ✅ 2026-04 ✅ 2026년3월 ❌ 3월 ❌
4
매달 선생님이 할 일
새 탭 추가 → 탭 이름을 2026-04로 변경 → Notion MASTER 컬럼 복사해서 A1에 붙여넣기
✅ 이후엔 앱에서 기준월 선택 → 🔄 클릭하면 자동으로 직전 6개월 데이터도 함께 로드돼요!
